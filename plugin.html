The Plugin I Chose

The plugin I chose is an arrow which appears when you scroll down the page.
Clicking the arrow brings you back to the top of the page, but if you scroll back up yourself, it disappears.

I chose this plugin because I thought it was very useful and relevant to any kind of website.
I picked it out from the JQuery plugin registry.

I followed the instructions on this website to implement the plugin in to my webpage. It can be seen on any page on my site.

The name of the function with the code in it is "toTopButton". Here is an example of what some of the code looks like:

$("body").toTopButton({
//path to icons
imagePath:'./img/icons',
arrowType:'arrow-circle',
iconColor:'b',
iconShadow: 6,
opacity: 20,
shape: 1,
margin: 3,
position:'br',
border : {
width: 0,
color:''
},


In this code, I am setting the image type and shape, and what position on the page it will be. I chose bottom right!
